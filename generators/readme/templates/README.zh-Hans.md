## 版本 (Versioning)

版本迭代遵循 SemVer 2.0.0 的规则。

但是，**当主版本号是零（0.y.z）**，一切**随时**可能**有不兼容的修改**。这处于开发初始阶段，其公共 API 是不稳定的。

关于 SemVer 的更多信息，请访问 http://semver.org/
